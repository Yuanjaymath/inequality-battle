# 一元一次不等式對戰遊戲(最基礎)

這是一個為國中七年級學生設計的 HTML5 對戰小遊戲，題目聚焦在一元一次不等式的最基礎概念，例如移項、加減法不等式、乘除法不等式與簡單解集判斷。

線上遊玩：<https://yuanjaymath.github.io/inequality-battle/>

## 遊戲特色

- 適合平板使用，左右兩側學生可以面對面快速對戰。
- 每位玩家有 7 條生命值，答對即可攻擊對方並扣除一條生命。
- 題型為四選一，降低操作負擔，讓學生專注在判斷不等式解集。
- 勝利時有花瓣雨與恭喜文字，失敗時也會出現鼓勵文字，營造正向學習氣氛。
- HTML、CSS、JavaScript 都整合在單一 `index.html` 檔案中，方便部署與修改。

## 教育理念

這個遊戲的核心目標不是取代正式教學，而是用短時間、高互動的方式提高學生練習意願。對許多七年級學生來說，一元一次不等式剛開始容易和等式運算混淆；透過即時作答、立即回饋與同儕對戰，可以讓學生在反覆判斷中建立熟悉度。

對戰機制能提供明確目標與情境感，讓練習不只是完成題目，而是變成一場需要專注、速度與準確度的活動。同時，遊戲中的鼓勵文字提醒學生：答錯不是失敗，而是下一次修正思考的機會。

## 適用情境

- 七年級一元一次不等式入門練習
- 課堂暖身活動
- 分組競賽
- 補救教學中的低壓力反覆練習
- 平板或互動螢幕上的雙人數學活動

## 使用方式

直接開啟 `index.html`，或使用上方 GitHub Pages 網址即可遊玩。建議使用平板橫放，兩位學生分別坐在左右兩側進行對戰。
